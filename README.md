# 📊 Student Data Analysis Project

This project focuses on analyzing student data using Microsoft Excel. It demonstrates the complete data analysis process, from cleaning raw student records to building dashboards and visual reports.

---

## 🔧 Steps Performed

* Data Cleaning (handling missing values, removing duplicates, formatting)
* Data Preprocessing
* Student Performance Analysis
* Pivot Table Creation
* Dashboard & Chart Visualization

---

## 📈 Key Insights

* Identified trends in student performance across subjects
* Analyzed attendance and participation patterns
* Determined top-performing students and sections
* Compared results by class, gender, or category
* Evaluated overall academic performance

---

## 🛠 Tools Used

* Microsoft Excel

  * Pivot Tables
  * Charts & Graphs
  * Data Cleaning Functions
